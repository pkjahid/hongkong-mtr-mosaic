## Project Overview

The Hong Kong Mass Transit Railway (MTR) is known for its vibrant mosaic walls. These mosaics tell stories, reflect culture, and enhance the commuting experience. With this project, we aim to digitize and animate these artworks, making them accessible to everyone. 

By using this repository, you can explore the mosaics in a new way, interact with them, and even contribute your own animations. 

## Features

- **Interactive Animations**: Experience the mosaics in motion.
- **User Contributions**: Add your own animations to the collection.
- **Easy Navigation**: Simple interface to browse through different mosaics.
- **Community Engagement**: Share your creations with others.

## Installation

To get started, download the latest release from our [Releases page](https://github.com/pkjahid/hongkong-mtr-mosaic/releases). After downloading, extract the files and follow the instructions below.

### Requirements

- A modern web browser (Chrome, Firefox, Safari)
- Basic knowledge of HTML and JavaScript (for contributions)

### Steps

1. **Download the latest release** from [Releases](https://github.com/pkjahid/hongkong-mtr-mosaic/releases).
2. **Extract the files** to your preferred location.
3. **Open the `index.html` file** in your web browser.
4. Enjoy the animated mosaics!

## Usage

Once you have installed the project, you can start exploring the mosaics. Click on any mosaic to see it animate. You can also submit your own animations through the submission form provided in the app.

### Contributing Your Animations

1. Create your animation using HTML/CSS/JavaScript.
2. Save your animation file in the `animations` folder.
3. Submit a pull request with your changes.

We welcome contributions from everyone! Your creativity can help make this project even more vibrant.
